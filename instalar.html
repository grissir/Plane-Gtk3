Instalar

Acontinuación describire los pasos para integrar la plantilla al sistema, espero en algun momento crear un paquete de instalación.

  • Tipografia
  • Theme Plane-Gtk3
  • Herramienta de retoques

1. Tipografia

  • * Descargamos la tipografia: Felipe fonts family
  • * Descomprimimos las fuentes descargadas en la carpeta:
    /home/usuario/.local/share/fonts/
    Nota: usuario lo remplazamos por el nombre de el usuario de tu sessión., recuerda que el directorio ".local" es oculto por si no lo encuentras debes habilitar: -mostar archivos ocultos- desde el menu -ver-.

2. Theme

  • * Descargamos el theme: Plane-Gtk3
  • * Descomprimimos el theme en:
    /home/usuario/.themes/
    Nota: usuario lo remplazamos por el nombre de el usuario de tu sessión., recuerda que el directorio ".themes" es oculto por si no lo encuentras debes habilitar: -mostar archivos ocultos- desde el menu -ver-. si el directorio .themes no esta creado puedes crearlo
